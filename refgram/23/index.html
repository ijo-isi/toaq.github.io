Incidental clauses

This section deals with the function word ju:

Pattern:
ju [sentence] → incidental clause

Incidental clauses are a means to place a sentence in a sentence to add incidental information about something in that sentence.

Parentheticals

Parentheticals allow arbitrary text (any number of complete or incomplete sentences) to be inserted inside a text, like English "(" and ")".

Pattern:
kıo [text] kı → parenthetical
